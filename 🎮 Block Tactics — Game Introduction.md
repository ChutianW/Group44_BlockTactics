# 🎮 Block Tactics — Game Introduction

Welcome to **Block Tactics**, a puzzle game where strategy and planning are the key to success.

---

## 🧩 What is Block Tactics?

Block Tactics is a **Sokoban-style puzzle game** played in the terminal.  
You control a character and push boxes onto their correct target positions.

It may look simple at first — but every move matters!

---

## 🎯 Your Goal

Push all boxes (`$`) onto the target positions (`^`).

When every target is covered by a box, **you win the level**.

---

## 🧱 Game Symbols

| Symbol | Meaning |
|--------|--------|
| `@` | You (the player) |
| `$` | Box (you can push it) |
| `^` | Target (where boxes should go) |
| `#` | Wall (cannot pass) |
| `.` | Obstacle (cannot pass) |
| ` ` | Empty space |

---

## 🎨 Colors (Visual Guide)

- 🟡 Player → easy to track  
- 🔴 Box → movable object  
- 🟢 Target → goal position  
- 🟢 Box on target → correct placement  

When a box reaches a target, it changes color — this means **progress!**

---

## 🎮 Controls

| Key | Action |
|-----|--------|
| W | Move Up |
| A | Move Left |
| S | Move Down |
| D | Move Right |
| R | Restart current level |
| Esc | Quit game |

---

## 📋 How to Play (Step-by-Step)

1. Start the game
2. Enter your username
3. Choose a difficulty level
4. Observe the map carefully
5. Plan your moves
6. Start pushing boxes toward targets
7. Complete all targets to win

---

## ⚠️ Important Rules

- You can only push **one box at a time**
- You **cannot pull** boxes
- You **cannot move through walls or obstacles**
- Boxes cannot pass through other boxes

---

## ❗ Be Careful: Common Mistakes

### 🟥 Corner Trap
If you push a box into a corner (not a target), it may get stuck forever.

### 🟥 Blocking Yourself
Pushing boxes in the wrong order can block your path.

### 🟥 Losing Access
You might accidentally block access to a target.

👉 Think before you move!

---

## 🔁 Stuck? No Problem

If you feel the level is impossible:

- Press **`R`** to restart the level
- The map stays the same
- Everything resets to the beginning

---

## 🎲 Difficulty Levels

| Level | Description |
|------|------------|
| Easy | Simple layout, good for beginners |
| Medium | More obstacles, requires planning |
| Hard | Complex layout, high challenge |

---

## 🧠 Tips to Win

- Plan several moves ahead  
- Keep space open for movement  
- Start with harder-to-reach boxes  
- Avoid pushing boxes against walls too early  

---

## 🚀 Ready to Start?

Take your time, think carefully, and enjoy solving the puzzle.

Good luck — and have fun playing **Block Tactics**! 🎯
